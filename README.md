# IoT Smart Helmet for Bikers 🚀

A smart safety system developed using ESP32 to enhance biker safety.

## 🔥 Features

* Helmet detection (bike won't start without helmet)
* Alcohol detection using MQ-3 sensor
* Accident detection using MPU6050 (impact + tilt detection)
* Motor control using relay
* Live monitoring dashboard using WiFi (WebServer)

## 🌐 Web Dashboard

* Real-time helmet status
* Alcohol detection status
* Motor ON/OFF control
* Accident alerts
* Sensor data (acceleration, tilt, temperature)

## ⚙️ Technologies Used

* ESP32
* Embedded C (Arduino)
* Sensors: IR, MQ-3, MPU6050
* WiFi + WebServer

## 💡 Working

The system ensures:

* Helmet must be worn
* No alcohol detected
* No accident detected

Only then the motor is enabled.
